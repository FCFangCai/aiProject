# 简单的ai应用聊天

## ✨ 包含功能

- 📄 **长期记忆**
- 🔍 **向量检索**
- 💬 **本地文档**
- 🎨 **上传文件 图转文**

## 🛠 技术栈

- **前端**: React+vite+ts
- **其他技术**
  - react-markdown:转换markdown
  - react-syntax-highlighter :代码高亮
  - remark-gfm:markdown支持table类型
  - @microsoft/fetch-event-source:流式传输

### 1. 克隆项目

```bash
git clone https://github.com/yourusername/ai-blog.git
cd ai-blog
```

### 2. 安装依赖

```bash
pnpm install
```
## 🚀 快速开始

```bash
npm run dev
```
